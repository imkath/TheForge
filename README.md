# TheForge

Micro-SaaS de detección de oportunidades de negocio impulsado por IA.

## Qué hace

Analiza señales del mercado para identificar oportunidades B2B emergentes. Procesa tendencias y patrones para sugerir nichos viables.

## Stack

- Cloudflare Workers
- IA para análisis de oportunidades

## Correr local

```bash
npm install
npm run dev
```
